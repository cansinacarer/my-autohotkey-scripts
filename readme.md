# My AutoHotKey Scripts

These keyboard macro scripts are compatible with AHK v2.

[AutoHotkey GitHub page](https://github.com/AutoHotkey/AutoHotkey)

[AutoHotkey Documentation](https://www.autohotkey.com/docs/v2/)

## Auto start scripts on boot
Run `shell:startup` and add a shortcut of the `.ahk` file in there.
